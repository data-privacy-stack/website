The Data Privacy Stack is a growing collection of open-source projects for
handling sensitive data. Each project keeps its own documentation, which you can
reach from its overview page.

## Presidio

**Detect and anonymize PII in text, images, and structured data.**

Presidio is a mature, extensible framework for identifying personally
identifiable information and transforming it — through redaction, masking,
replacement, encryption, and synthetic surrogates.

- [Presidio overview](/projects/presidio/)
- [Documentation](https://presidio.dataprivacystack.org)
- [GitHub](https://github.com/data-privacy-stack/presidio)
